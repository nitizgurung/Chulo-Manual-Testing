# Chulo Manual Testing

This repository contains comprehensive manual test scenarios and test cases for the Chulo e-commerce platform. The test documentation covers critical user workflows including checkout, product searching, order confirmation, and user reviews.

## Project Structure

```
Chulo Manual Testing/
├── excel_sheets/          # Test cases in Excel format (.xlsx)
├── pdfs/                  # Test cases in PDF format (.pdf)
└── README.md
```

## Test Scenarios & Cases

This project includes test documentation for the following features:

1. **Check out processing** - Test cases for the checkout workflow and payment processing
2. **Edit And Delete Reviews** - Test cases for managing user reviews (editing and deletion)
3. **Order Confirmation** - Test cases for order confirmation and related processes
4. **Searching and Filtering** - Test cases for product search and filtering functionality
5. **Submit Reviews and Ratings** - Test cases for submitting product reviews and ratings
