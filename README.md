# Chulo Manual Testing

This repository contains comprehensive manual test scenarios and test cases for the Chulo e-commerce platform. The test documentation covers critical user workflows including checkout, product searching, order confirmation, and user reviews.

## Project Structure

```
Chulo Manual Testing/
├── excel_sheets/          # Test cases in Excel format (.xlsx)
├── pdfs/                  # Test cases in PDF format (.pdf)
└── README.md
```

## Test Scenarios & Cases

This project includes test documentation for the following features:

1. **[Check out processing](pdfs/Test%20Scenario%20and%20Test%20Case%20for%20Check%20out%20processing.pdf)** - Test cases for the checkout workflow and payment processing
2. **[Edit And Delete Reviews](pdfs/Test%20Scenario%20and%20Test%20Case%20for%20Edit%20And%20Delete%20Reviews.pdf)** - Test cases for managing user reviews (editing and deletion)
3. **[Order Confirmation](pdfs/Test%20Scenario%20and%20Test%20Case%20for%20Order%20Confirmation.pdf)** - Test cases for order confirmation and related processes
4. **[Searching and Filtering](pdfs/Test%20Scenario%20and%20Test%20Case%20for%20Seaching%20and%20Filtering.pdf)** - Test cases for product search and filtering functionality
5. **[Submit Reviews and Ratings](pdfs/Test%20Scenario%20and%20Test%20Case%20for%20Submit%20Reviews%20and%20Ratings.pdf)** - Test cases for submitting product reviews and ratings
